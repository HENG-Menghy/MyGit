The border shorthand

This property specifies to display the use of border shorthand:

A dotted, 1px, red border.

A dashed, 2px, blue border.

A solid, 3px, aqua border.

A double, 4px, powderbule border.

A groove, 5px, gray border.

A ridge, 6px, green border.

An inset, thin, pink border.

An outset, medium, hotpink border.

No border style.

A hidden border.

A mixed border.
